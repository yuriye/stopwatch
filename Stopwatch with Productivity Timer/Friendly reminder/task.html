Description

A notification is a good way to remind your users that the application is there! With this feature, we will make our app more convenient and user-friendly. The user will receive a notification when the stopwatch reaches the upper limit but will still be able to keep working and track the overall elapsed time.

To create a simple notification, read up on working with Android notifications. Note that setSmallIcon, setContentTitle, and setContentText are required for creating a notification.

Objectives

Add an Android notification after the stopwatch reaches the upper limit.

Use the following values:

  • org.hyperskill for the notification channel id;
  • 393939 for the notification id.
